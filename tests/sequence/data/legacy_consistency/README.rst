Legacy consistency data
======================

This directory contains precomputed pairwise alignments of the Cas9 sequences
from ``tests/sequence/data/cas9.fasta``, generated by the legacy *Cython*
implementations.
These files are used by consistency tests to verify that newer Rust implementations
produce identical results.

The files were created by ``create_legacy_alignments.py`` in this directory.

Files
-----

- ``align_optimal_<i>_<j>.fasta`` — global alignment of sequence *i* and *j*
  using ``align_optimal()`` with BLOSUM62.
- ``align_banded_<i>_<j>.fasta`` — banded alignment of sequence *i* and *j*
  using ``align_banded()`` with a band derived from the optimal alignment.
- ``align_local_<i>_<j>.fasta`` — local gapped alignment of sequence *i* and
  *j* using ``align_local_gapped()`` with a seed derived from the optimal
  alignment.
- ``params.json`` — alignment parameters (gap penalty, band, seed, threshold)
  used to generate each file, keyed by function name and sequence pair.
